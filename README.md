# LoginRegisterSystem
## 功能描述
 1. 具有基本的登陆注册功能
 2. 人性化的交互
 3. 利用正则表达式判断用户注册是否正确
## 运行环境
javaEE + tomcat + mysql +
jsp + servlet + bootstrap + jquery
## 所需Jar包
mysql-connector-java-5.0.8-bin.jar
